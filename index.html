FoundIt! - Lost & Found Web App

In my sixth semester project, I developed FoundIt, a web application that fosters a community-driven approach to locating lost items. Recognizing the frustration of misplaced belongings, I designed FoundIt to empower users by connecting them with others who might have encountered their missing treasures. The application leverages ReactJS to create a user-friendly and interactive front-end, while Firebase provides a robust back-end infrastructure to manage user data and interactions. Additionally, FoundIt utilizes Leaflet.js, a powerful JavaScript library, to integrate map functionalities, allowing users to pinpoint the location where their items were last seen. Through FoundIt, I hope to cultivate a collaborative environment where communities can assist each other in the often-stressful task of recovering lost belongings.
